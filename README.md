# CRUD приложение API
Сервис позволяет получать информацию, создавать, изменять и удалять пользователя.
# Установка
1. Устанавливаем пакеты docker engine и docker compose.
2. Копируем репозиторий из git
3. Чистим директорию `postgres` в корневой директории командой `rm -rf ./postgres/.gitkeep`
4. Запускаем сборку и запуск контейнеров командой `docker compose up`
## Использование
Приложение будет доступно по ссылке `http://localhost:8080/`.
Для проверки работоспособности, перейдите по ссылке`http://localhost:8080/api/v1/users`.
